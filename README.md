# 玄机子命盘 - 八字排盘小程序

一个微信小程序，输入出生年月日时，自动排出八字命盘。

写这个的起因很简单：我妈在家族群里天天转发那些手工排盘的截图，字迹模糊得跟心电图似的。我说我给你写个小程序吧，三秒钟出结果，比手画快多了。

## 做了什么

输入公历出生日期和时间，自动换算成农历干支，排出四柱八字（年柱、月柱、日柱、时柱），同时显示十神、纳音、五行等信息。

排盘逻辑跑在云函数上，核心算法用 Python 写的，挂在微信云托管。前端就是普通的小程序页面，没什么花活——输入表单进去，命盘表格出来。

## 技术栈

- 前端：微信小程序原生框架（WXML + WXSS + JS）
- 后端：微信云函数（Node.js）+ 云托管（Python 排盘引擎）
- 排盘算法：基于《渊海子平》和现代万年历数据交叉校准

## 目录结构

```

├── miniprogram/           # 小程序前端
│   ├── app.js
│   ├── app.json
│   ├── app.wxss
│   └── pages/paipan/      # 排盘页面
├── cloudfunctions/
│   └── baziPaipan/        # 排盘云函数
├── cloudrun/
│   └── bazi-engine/       # Python 排盘引擎

```

## 怎么跑

1. 用微信开发者工具打开 `miniprogram/` 目录
2. 配置云开发环境，上传 `cloudfunctions/` 下的云函数
3. 云托管部署 `cloudrun/bazi-engine/` 的 Python 引擎
4. 编译预览

## 已知问题
