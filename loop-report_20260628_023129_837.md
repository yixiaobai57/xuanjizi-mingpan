## 阻塞性 Bug 清单

### Bug 1 [CRITICAL] paipan.py L84 — 性别参数反转
```python
yun = bazi_obj.getYun(gender == 'male' and 0 or 1)
```
**原因**: Python 布尔表达式求值陷阱。`gender == 'male' and 0 or 1` 当 gender='male' 时结果为 1（应为 0），导致男性用户被当女性处理大运排盘。
**影响**: 男性用户大运起点、起运年龄、大运顺逆全部计算错误。
**修复**: 改为 `0 if gender == 'male' else 1`。

### Bug 2 [CRITICAL] 缺少 __init__.py 导致相对导入失败
```python
# paipan.py L5
from .utils import (TIAN_GAN, DI_ZHI, CANG_GAN, ...)
```
**原因**: `cloudrun/bazi-engine/` 目录下无 `__init__.py`，Python 不将其视为包，相对导入 `from .utils` 将抛出 `ImportError: attempted relative import with no known parent package`。
**影响**: 服务启动即崩溃，无法运行。
**修复**: 在 `cloudrun/bazi-engine/` 下创建空 `__init__.py`。

### Bug 3 [HIGH] Dockerfile 入口指向旧 main.py
```dockerfile
CMD ["gunicorn", "-w", "2", "-b", "0.0.0.0:8080", "main:app"]
```
**原因**: `main.py` 仍是 v0.1.0 mock 代码（返回 "not implemented yet"），v0.2.0 实现在 `main_20260628_020956_571.py` 中。
**影响**: 部署到云托管后，/api/paipan 返回 mock 结果而非真实排盘。
**修复**: 将 `main_20260628_020956_571.py` 覆盖写入 `main.py`，确保 v0.2.0 代码生效。
